title: 自定义 API 密钥
---

### OpenAI API 密钥

Cursor 允许您输入自己的 OpenAI API 密钥，以便按自己的费用发送任意数量的 AI 消息。

您可以从 [这里](https://platform.openai.com/account/api-keys) 获取自己的 API 密钥。要使用自己的 API 密钥，请转到 `Cursor 设置` > `模型` > `OpenAI API 密钥` 并输入您的 API 密钥。然后，点击“验证”按钮。一旦您的密钥被验证，您的 OpenAI API 密钥将被启用。

### Google API 密钥

对于 Google API 密钥，您可以设置自己的 API 密钥，以便按自己的费用使用 Google 模型，例如 `gemini-1.5-flash-500k`。

### 我的 API 密钥会被存储或离开我的设备吗？

您的 API 密钥不会被存储，但它会在每个请求中发送到我们的服务器。所有请求都通过我们的后端路由，因为我们在这里进行最终的提示构建。
