<Info>目前，`@Git` 仅在 Cursor 聊天中支持。</Info>

### `@Git`

在 Cursor 的聊天中，您可以使用 `@Git` 将 git 提交、差异或拉取请求添加到您的提示中。

<Frame>
  <img src="/images/context/@git.png" />
</Frame>

### 常见用例

`@Git` 的一个常见用例是允许 Cursor 的 AI 扫描差异并查找可能由差异引起的错误或问题。

<Frame>
  <img src="/images/context/@git-usecase1.png" />
</Frame>

您还可以使用 `@Diff of Working State` 从当前差异生成提交消息。

<Frame>
  <img src="/images/context/@git-commit-message.png" />
</Frame>
