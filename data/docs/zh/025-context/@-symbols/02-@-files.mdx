### `@Files`

在 Cursor Chat 和 Cmd K 等 AI 输入框中，您可以通过使用 `@Files` 来引用整个文件。
此外，如果您在 `@` 后继续输入，您将在 [`@代码`](/context/@-symbols/@-code) 策略后看到文件搜索结果。

为了确保您引用的文件是正确的，Cursor 将显示文件路径的预览。这在您有多个同名文件位于不同文件夹时尤其有用。

### 聊天长文件引用

在 Cursor 的聊天中，如果文件内容过长，Cursor 将把文件分成较小的块，并根据与查询的相关性对其进行重新排序。

### Cmd K 分块策略

对于 Cmd K，Cursor 也根据内容长度以不同方式使用文件引用。

- 自动
  - 根据文件大小自动选择三种阅读策略之一
- 完整文件
  - 整个文件作为上下文使用。
- 大纲
  - Cursor 解析文件的大纲并将信息用作上下文。
- 块
  - Cursor 将文件分成较小的块，并选择最相关的一个。

<Frame>
  <img src="/images/context/@file-cmdk.png" />
</Frame>

### 拖放

您可以将文件从主侧边栏拖放到聊天或 Cmd K 中，以将其作为上下文添加。
