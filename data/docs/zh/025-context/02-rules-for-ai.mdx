---
title: 人工智能规则
---

You can add custom instructions to Cursor by modifying the `人工智能规则` section under `Cursor Settings` > `General` > `人工智能规则`.

This custom instruction will be included for features such as Cursor Chat and Ctrl/⌘ K.

<Frame>
  <img src="/images/context/rules-for-ai.png" />
</Frame>

### `.cursorrules`

For project-specific instructions, you can include the instructions in a `.cursorrules` file in the root of your project.

As the same as the "人工智能规则" section, the instructions in the `.cursorrules` file will be included for features such as Cursor Chat and Ctrl/⌘ K.
